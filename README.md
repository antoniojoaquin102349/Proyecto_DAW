# Proyecto_DAW

Proyecto de tienda online de piezas para todoterrenos, desarrollado con PHP y HTML. Permite a los usuarios registrarse, iniciar sesión (incluyendo Google), gestionar su carrito de compras y consultar sus pedidos.

Tecnologías

Backend: PHP

Frontend: HTML, CSS, JavaScript

Base de datos: MySQL / MariaDB

Autenticación: Inicio de sesión tradicional y Google OAuth

Funcionalidades

Registro de usuario y autenticación tradicional

Inicio de sesión con Google

Visualización de productos y búsqueda

Añadir productos al carrito de compras

Guardar y consultar pedidos

Gestión de perfil de usuario

Instalación

Clonar el repositorio:

git@github.com:antoniojoaquin102349/Proyecto_DAW.git

Configuración

Configurar tu servidor local (XAMPP, MAMP o similar)

Importar la base de datos desde el archivo .sql incluido en el proyecto

Ajustar las credenciales de la base de datos en el archivo config.php o equivalente

Abrir el proyecto en el navegador (http://localhost/nombre-del-proyecto)

Uso

Registrarse o iniciar sesión (incluyendo Google)

Explorar productos y añadirlos al carrito

Confirmar pedidos

Revisar historial de pedidos en el perfil

Estructura
/css        # Estilos
/js         # Scripts de JavaScript
/images     # Imágenes del proyecto
/backend    # Archivos PHP y lógica del servidor
